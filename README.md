

# json2image

通用的静态图片生成解决方案。

使用场景：

* H5 页面生成分享海报。
* 小程序下载分享图片。
* 小程序自定义分享卡片。


## 特点

* 跨端，支持浏览器和 node 环境。


## 使用

### 安装

这个库在前端不依赖第三方库，在 node 端依赖 node-canvas。


前端 CDN 引入：

```js

```

前端 npm 引入

```js
import json2iamge from '@yunser/json2image'
```

Node.js 引入

```js
const json2iamge = require('@yunser/canvas-node')
```
